# Airnbn: Fullstack Web Application

Airnbn is an Airbnb clone for renting rooms and apartments. It was designed by a team of four people during the Le Wagon Web Development bootcamp.


![airnbn](https://github.com/Jide-Muritala/AirNBN_Clone_MS/assets/115728688/9f543211-c5eb-4282-82b9-ee08d0103a77)


The app is built on Ruby on Rails. It uses PostgreSQL for the database, Devise for user authentication, and Pundit for authorization. 
It uses Mapbox for location mapping and Cloudinary for user media management.

rails s
